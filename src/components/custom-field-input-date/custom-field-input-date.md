##### Basic usage:

```jsx
<CustomFieldInputDate
    id="my-birthday-1"
    label="Andre's Birthday" 
    value="1992-05-10"
/>
```
----

##### Disabled state:

```jsx
<CustomFieldInputDate
    id="my-birthday-4"
    label="Juanca's Birthday"
    value="03/03/1990" 
    disabled
/>
```
----

##### Errored state:

```jsx
<CustomFieldInputDate
    id="not-my-birthday"
    label="Andre's Birthday" 
    helpText="This is not my birthday"
    value="01/01/0001"
    error
/>
```
----
